### Dapr Agents v1.0 GA — 云原生 AI Agent 框架进入生产

在 KubeCon EU 2026 上，CNCF 宣布 Dapr Agents v1.0 正式 GA。这是基于 Dapr 分布式运行时构建的 Python 框架，专为在 Kubernetes 上运行生产级 AI Agent 设计。核心能力：持久化工作流（自动重试 + 故障恢复）、跨 30+ 数据库的状态管理、SPIFFE 身份安全、多 Agent 协调、内置可观测性。ZEISS Vision Care 已在生产使用。关键：可以无代码改动切换 LLM provider。

🔗 Source: [CNCF](https://www.cncf.io/announcements/2026/03/23/general-availability-of-dapr-agents-delivers-production-reliability-for-enterprise-ai/)

🎯 Action: 这跟你的 Andromeda 项目 Week 8-10 Agent 模块直接相关。Dapr 是微软发起的项目，你作为微软 Tech Lead 有天然优势。花 20 分钟读 [Dapr Agents 文档](https://docs.dapr.io/developing-ai/dapr-agents/dapr-agents-introduction/)，评估是否可以替换 Andromeda 里的 Agent 编排层。下次 standup 可以提：「Dapr Agents GA 了，我们的项目可以考虑集成」。

### GPT-5.4 实测：多步骤任务成功率 83%，原生 Computer Use

OpenAI GPT-5.4（3/5 发布）的第三方评测数据陆续出来。GDPval 基准测试成功率 83.0%（GPT-5.2 为 70.9%），法律文档基准 91%。最大亮点：原生 computer-use — 模型可以解读截图并发出鼠标/键盘指令，让 Agent 自动操作浏览器和软件。Token 消耗比 5.2 大幅降低。API 已开放 gpt-5.4 和 gpt-5.4-pro。

🔗 Source: [RiskInfo AI](https://www.riskinfo.ai/post/ai-insights-key-global-developments-in-march-2026) | [OpenAI](https://openai.com/index/introducing-gpt-5-4/)

🎯 Action: Computer Use 是 Agent 方向的重大进展。Andromeda Week 9 如果涉及 browser automation Agent，GPT-5.4 的原生能力可以替代 Playwright + 截图 → LLM 的笨办法。在 Andromeda repo 记个 issue：评估 GPT-5.4 computer-use API。

### TypeScript 6.0 正式发布 — 最后一个 JavaScript 编译器版本

微软正式发布 TypeScript 6.0（从 RC 进入 stable）。这是基于 JavaScript 代码库的最后一个大版本 — TypeScript 7.0 将用 Go 重写（原生编译 + 多线程共享内存），编译速度预计提升 10x+。TS 6.0 是 5.9 → 7.0 的桥梁版本。

--moduleResolution node
废弃，迁移到
nodenext
或
bundler
；④ import assertion 语法废弃。

🔗 Source: [TypeScript Blog](https://devblogs.microsoft.com/typescript/announcing-typescript-6-0-rc/)

🎯 Action:

🟡 KNOW

### Excel Copilot XSS 漏洞 (CVE-2026-26144) — 新型 AI 攻击面

Patch Tuesday 中一个值得特别关注的漏洞：CVE-2026-26144 是 Excel 中的 Critical XSS，攻击者可利用它让 Copilot Agent 泄露数据。这是一种新型攻击路径 — 通过注入恶意内容到文档，诱导 AI Agent 执行非预期操作。另外两个 Office RCE（CVE-2026-26110/26113）在 Preview Pane 中可触发。

🔗 Source: [Help Net Security](https://www.helpnetsecurity.com/2026/03/11/march-2026-patch-tuesday/)

🎯 Action: AI Agent 安全是新兴领域。这个 Copilot XSS 是教科书级的 prompt injection via document 案例。记到 Andromeda 项目笔记里 — 当你构建 AI Agent 时，文档输入是攻击面。确保你的 Windows Update 最新。
